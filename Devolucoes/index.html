Configuração

Para adicionar, editar e excluir registros, é necessário um Token do GitHub com permissão de escrita no repositório.

Como criar o token:
1. Acesse github.com/settings/tokens
2. Clique em "Generate new token" (Fine-grained)
3. Em Repository access, selecione "Only select repositories" → Wdc-Ajudas
4. Em Permissions → Contents, marque "Read and write"
5. Gere e copie o token
