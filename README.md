## Português

### 📝 Descrição
O **WDC Ajudas** é uma ferramenta interna desenvolvida para otimizar as operações diárias da equipe, padronizando a comunicação e automatizando tarefas repetitivas. O objetivo principal é aumentar a produtividade e reduzir erros em processos de suporte, vendas e logística.

### ✨ Funcionalidades Principais

#### 📧 Scripts de E-mails
Uma central de templates inteligentes para diversos setores:
* **SAC:** Devoluções (RMA), Trocas em Garantia, Solicitação de Entrada de NF e Advanceds.
* **Apoio a Vendas:** Relatórios de atraso, pendências com transportadoras, notificações de NFs retidas e emissão de tickets Correios.
* *Diferencial:* Preenchimento dinâmico de campos com formatação automática e sistema de cópia para área de transferência com fundo branco para compatibilidade com Outlook/Gmail.

#### ⚡ Automações & Atalhos
Ferramentas para processamento rápido de dados:
* **Comparar Lista:** Validação de divergências entre listas de itens.
* **Verificar Garantia:** Consulta rápida de prazos e políticas (incluindo regras específicas para revendedores).
* **Análise de NF:** Ferramentas para conferência de valores unitários e validação de notas fiscais.
* **Gerador de Tabelas:** Criação de estruturas de dados para documentação.

#### 🔍 Central de Pesquisas & Processos
Guias passo a passo e ferramentas de busca para os processos internos:
* **Consultas:** Pesquisa por Serial, LVP, NF de Remessa e Serial KIT.
* **Procedimentos:** Como baixar Notas Fiscais, abrir Ordens de Serviço (OS) e visualizar Romaneios no sistema Visão.

### 🛠️ Tecnologias Utilizadas
* HTML5 / CSS3 (Design Responsivo)
* JavaScript Vanilla (Lógica de Automação e Manipulação de DOM)
* Ícones e Identidade Visual WDC

## English

### 📝 Description
**WDC Ajudas** is an internal tool developed to optimize the team's daily operations by standardizing communication and automating repetitive tasks. The main goal is to increase productivity and reduce errors in support, sales, and logistics processes.

### ✨ Key Features

#### 📧 Email Scripts
A hub of smart templates for various departments:
* **Customer Service (SAC):** Returns (RMA), Warranty Exchanges, Invoice Entry Requests, and Advanced processes.
* **Sales Support:** Delay reports, carrier pending issues, withheld invoice notifications, and Postal Service (Correios) ticket issuance.
* *Highlight:* Dynamic field filling with automatic formatting and a "Copy to Clipboard" system with a white background for Outlook/Gmail compatibility.

#### ⚡ Automations & Shortcuts
Tools for fast data processing:
* **List Comparison:** Validation of discrepancies between item lists.
* **Warranty Checker:** Quick consultation of terms and policies (including specific rules for resellers).
* **Invoice Analysis:** Tools for checking unit values and validating fiscal invoices.
* **Table Generator:** Creation of data structures for documentation.

#### 🔍 Research & Process Hub
Step-by-step guides and search tools for internal processes:
* **Queries:** Search by Serial Number, LVP, Shipping Invoices, and KIT Serials.
* **Procedures:** How to download Invoices, open Service Orders (OS), and view Packing Slips in the "Visão" system.

### 🛠️ Technologies Used
* HTML5 / CSS3 (Responsive Design)
* Vanilla JavaScript (Automation Logic and DOM Manipulation)
* WDC Visual Identity & Icons
