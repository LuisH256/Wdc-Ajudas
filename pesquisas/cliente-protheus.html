Pesquisa Cliente (Código/Contato) - Passo a Passo

1. Acessar Cadastro de Clientes

  • Cadastro
    • Atualização
      • Clientes

2. Encontrar o Cliente pelo CNPJ (Visão)

  • Utilize ferramenta de pesquisa (caso o Visão esteja funcionando pode pequisar o CNPJ e pegar o codigo) dentro da tela de Clientes.
  • Busque o cliente utilizando o CNPJ.
  • Verifique se a Razão Social no Protheus corresponde ao cliente desejado.
  • Anote o Código do Cliente.

3. Obter Contato para Transportadora

  • Com o cliente selecionado, vá em Outras Ações.
  • Selecione a opção Contatos.
  • Nesta tela, você encontrará as informações de contato para serem repassadas, como o Nome do Contato e Telefone (ou informações específicas necessárias para a transportadora).
