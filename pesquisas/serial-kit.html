Pesquisa Serial KIT (Módulo 04)

1. Acessando a Consulta Genérica

  • Consultas
    • Cadastros
      • Genéricos

2. Configuração e Busca por PA

  • Na tela de Consultas Genéricas, digite WA8 e clique em OK.
  • (Importante: É necessário ter o PA do equipamento. Verifique como pesquisar o PA em: Pesquisa Serial
  • Vá nas 3 barras de Opções e use o filtro Número Série.

    Dependendo do seu acesso, utilize o formato: Filial + Número de série.

  • Coloque o PA e pesquise.

3. Obtendo o ID INTEGRA

  • Após localizar o registro, dê dois cliques para abrir.
  • Localize a caixa ID INTEGRA, que fica no campo inferior direito.
  • Copie o código.

4. Pesquisa Final com ID INTEGRA

  • Feche a tela de pesquisa de detalhes.
  • Vá nos 3 pontos de Opções novamente e coloque o filtro Id Integrac.

    Dependendo do seu acesso, utilize o formato: Filial + Id Integrac.

  • Cole o código Id Integra copiado e pesquise.
