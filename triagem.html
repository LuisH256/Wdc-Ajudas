Instruções para Triagem de Equipamentos

Siga este fluxo para realizar a triagem dos equipamentos inseridos na planilha "RELAÇÃO DE PRODUTOS WDC NETWORK".

1. Consulta no Protheus (Módulo 04)

Utilize os seriais fornecidos pelo cliente para localizar as informações de cadastro:

  • Módulo Componentes: Pesquise o serial para encontrar o Código SN e o PA.
    ➔ Clique aqui para ver o passo a passo

  • Módulo Produto Acabado: Verifique o histórico de romaneio para identificar a Data de Garantia e NF.
    ➔ Clique aqui para ver o passo a passo
Regra de Cálculo (Exclusiva Protheus):
• Garantia Padrão: Data encontrada + 1 ano.
• Se for Revendedor: Data encontrada + 1 ano e 3 meses.

2. Triagem com Automações (Serial VL)

A automação Serial VL é a principal forma de triagem em massa via Microsoft Teams:

  • Clique com o botão direito em uma conversa > Serial VL.
  • Clique em Abrir Planilha e cole os seriais usando CTRL + SHIFT + V.
  • O link de resultado trará os dados que devem ser colados na aba TRIAGEM.
Identificação de Operação via CFOP:
Ao analisar o resultado do Serial VL, identifique a natureza da operação:
• Venda: CFOP 5101 ou 6101.
• Locação: CFOP 5949 ou 6949.
• Conserto: Localizar NF original de remessa no Protheus.
Atenção: Para datas coletadas via automação, siga a data informada normalmente (sem somar 1 ano), exceto para produtos Fujikura.

REVENDEDOR (Fora de Garantia):
• Se for VENDA: Adicione +3 meses extras.
• Se for LOCAÇÃO: Não adicionar os 3 meses.

4. Padronização e SharePoint

Nomenclatura da NF:

  • Filial Padrão: Adicionar 000 antes do número (Ex: 000921421).
  • Filial 08: Adicionar 0000 antes do número (Ex: 000021521).

Fluxo de Salvamento:

  • Suba o arquivo em: Suporte & Garantia ➔ Notas ➔ NFs Triagem.
  • Clique em Compartilhar ➔ Ícone da Engrenagem ⚙️.
  • Selecione "Somente pessoas com acesso existente" e clique em Aplicar.
  • Copie o link gerado.

Vínculo na Planilha:

  • Na célula da NF na planilha de triagem, use CTRL + K e cole o link do SharePoint.
