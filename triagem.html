Instruções para Triagem de Equipamentos

Siga este fluxo para realizar a triagem dos equipamentos inseridos na planilha "RELAÇÃO DE PRODUTOS WDC NETWORK".

1. Consulta no Protheus (Módulo 04)

Utilize os seriais fornecidos pelo cliente para localizar as informações de cadastro:

  • Módulo Componentes: Pesquise o serial para encontrar o Código SN e o PA.
    ➔ Clique aqui para ver o passo a passo

  • Módulo Produto Acabado: Verifique o histórico de romaneio para identificar a Data de Garantia e NF.
    ➔ Clique aqui para ver o passo a passo
Regra de Cálculo (Exclusiva Protheus):
• Garantia Padrão: Data encontrada + 1 ano.
• Se for Revendedor: Data encontrada + 1 ano e 3 meses.

2. Triagem com Automações e CFOP

A automação Serial VL (Teams) traz os dados brutos para a aba TRIAGEM. Use-a para identificar a operação:

  • Clique com o botão direito em uma conversa > Serial VL.
  • Cole os seriais usando CTRL + SHIFT + V.
Análise de Operação via CFOP:
• Venda: CFOP 5101 ou 6101.
• Locação: CFOP 5949 ou 6949.
• Conserto: Localizar NF original de remessa no Protheus.
Atenção: Para datas via automação, siga o que for informado, exceto para Fujikura.

4. Padronização e SharePoint

Nomenclatura da NF: Adicione 000 (Padrão) ou 0000 (Filial 08) antes do número.

Caminho: Suporte & Garantia ➔ Notas ➔ NFs Triagem.

Compartilhamento: ⚙️ Engrenagem ➔ "Somente pessoas com acesso existente" ➔ Aplicar ➔ Copiar Link.

Na Planilha: Use CTRL + K para inserir o link.

5. Modelo de Nota (Geração da NF)

Após concluir a triagem técnica, você deve montar o arquivo que servirá de base para a emissão da nota.

⚠️ IMPORTANTE: Solicite o arquivo base do "Modelo de Nota para Triagem" ao Supervisor (Angela).

Regras de Preenchimento:

  • Código do Produto: Se o item for Produto Acabado, use o código PA. Se for Material de Engenharia, use o código ME.
  • Consolidação por EAN: Caso existam modelos iguais com EANs diferentes, liste um de cada.
  • Quantidade: Realize a somatória total de todos os produtos que possuem o mesmo EAN e insira o resultado no campo.
  • Valor Unitário: Use o valor de uma nota fiscal de aquisição daquele EAN (priorize os dados da 1ª NF encontrada).
  • Solicitação/CRG: No campo de Solicitação de reparo/garantia, insira o número do CRG.
  • Histórico de Notas: No campo "Nota fiscal de aquisição", liste os números de todas as NFs vinculadas àqueles itens.
