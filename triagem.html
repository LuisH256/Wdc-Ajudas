Instruções para Triagem de Equipamentos

Siga este fluxo para realizar a triagem dos equipamentos inseridos na planilha "RELAÇÃO DE PRODUTOS WDC NETWORK".

1. Consulta no Protheus (Módulo 04)

Utilize os seriais fornecidos pelo cliente para localizar as informações de cadastro:

  • Módulo Componentes: Pesquise o serial para encontrar o Código SN e o PA.
  • Módulo Produto Acabado: Verifique o histórico de romaneio para identificar a Data de Garantia e NF.
Regra de Cálculo: Padrão (+1 ano) | Revendedor (+1 ano e 3 meses).

2. Triagem com Automações (Serial VL)

Use o Serial VL para processar os dados e alimentar a aba TRIAGEM.

Identificação de Operação via CFOP:
• Venda: 5101 ou 6101 | Locação: 5949 ou 6949.

3. Tabela de Garantia por Fabricante

Fabricante / ProdutoTempo TotalAção
Fujikura3 AnosAdicionar +2 anos.
Grandstream1 a 3 AnosValidar conforme modelo.

4. Padronização e SharePoint

  • Nomenclatura: Adicionar 000 (Padrão) ou 0000 (Filial 08) antes da NF.
  • Link: Engrenagem ⚙️ > "Somente pessoas com acesso existente".

5. Elaboração do Modelo de Nota

Após finalizar a triagem, você deve consolidar as informações para criar o modelo de nota fiscal.

🚨 ATENÇÃO - PRIMEIRA VEZ: Se esta é a sua primeira triagem ou você ainda não possui o arquivo base, é obrigatório solicitar o Modelo de Nota para Triagem diretamente à Supervisora (Angela).

Regras de Preenchimento:

  • 📌 Código do Produto: Diferencie entre PA (Produto Acabado) e ME (Componente). Use o código correspondente ao tipo de entrada.
  • 📌 EAN e Descrição: Se houver múltiplos EANs, liste uma linha para cada (apenas um EAN de cada).
  • 📌 Quantidade: Realize a somatória total de todos os produtos que possuem o mesmo EAN.
  • 📌 Valor Unitário: Use o valor da NF de aquisição. Havendo vários, priorize o valor da 1ª Nota Fiscal daquele EAN.
  • 📌 N° Solicitação: Preencher obrigatoriamente com o número do CRG.
  • 📌 Notas Fiscais: No campo de aquisição, cite todos os números das NFs envolvidas.
