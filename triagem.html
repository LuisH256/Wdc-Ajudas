Instruções para Triagem de Equipamentos

1. Consulta no Protheus (Módulo 04)

Utilize os seriais fornecidos pelo cliente para localizar as informações de cadastro:

  • Módulo Componentes: Pesquise o serial para encontrar o Código SN e o PA.
  • Módulo Produto Acabado: Verifique a Data de Garantia e NF.
Cálculo: Padrão (+1 ano) | Revendedor (+1 ano e 3 meses).

2. Triagem com Automações (Serial VL)

Use o Serial VL no Teams para processar em massa e alimentar a aba TRIAGEM.

Identificação de Operação via CFOP:
• Venda: 5101 / 6101
• Locação: 5949 / 6949
• Conserto: Localizar NF de remessa original.

4. Padronização e SharePoint

Nomenclatura (Zeros): Filial Padrão (000) | Filial 08 (0000).

Link: ⚙️ Engrenagem ➔ "Somente pessoas com acesso existente".

5. Criação do Modelo de Nota

O modelo de nota deve ser gerado obrigatoriamente a partir dos dados consolidados na triagem.

⚠️ Importante: Solicitar ao Supervisor (Angela) o arquivo oficial do modelo de nota para triagem.

Campos que devem ser alterados:

  • Código do Produto: Se o item for PA, use o código PA. Se for ME, use o código ME.
  • Descrição e EAN: Se houver mais de um EAN para o mesmo modelo, insira uma linha para cada (apenas um EAN de cada tipo).
  • Quantidade: Faça a somatória total de todos os produtos pertencentes a cada EAN específico.
  • Valor Unitário: Utilize o valor de uma das Notas Fiscais do EAN correspondente (Prioridade: valor da 1ª NF encontrada).
  • Solicitação de Reparo/Garantia: Inserir o número do CRG.
  • Nota Fiscal de Aquisição: Listar os números de todas as Notas Fiscais relacionadas àqueles itens.
