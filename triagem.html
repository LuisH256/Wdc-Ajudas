Instruções para Triagem de Equipamentos

1. Consulta no Protheus (Módulo 04)

Utilize os seriais fornecidos pelo cliente para localizar as informações de cadastro:

  • Módulo Componentes: Pesquise o serial para encontrar o Código SN e o PA.
  • Módulo Produto Acabado: Verifique a Data de Garantia e NF.
Regra de Cálculo: Padrão (+1 ano) | Revendedor (+1 ano e 3 meses).

2. Triagem com Automações (Serial VL)

Use o Serial VL no Teams para processar em massa e alimentar a aba TRIAGEM.

Identificação de Operação via CFOP:
• Venda: 5101 / 6101
• Locação: 5949 / 6949
• Conserto: Localizar NF de remessa original.

4. Padronização e SharePoint

Nomenclatura (Zeros): Filial Padrão (000) | Filial 08 (0000).

Link SharePoint: ⚙️ Engrenagem ➔ "Somente pessoas com acesso existente".

5. Criação do Modelo de Nota

O modelo de nota deve ser preenchido criteriosamente com base nos dados consolidados na triagem anterior.

📋 Procedimento com Supervisão Se esta for a sua primeira vez realizando este processo, você deve solicitar o arquivo oficial do modelo de nota diretamente à supervisora Angela.

Orientações de Preenchimento:

Código do Produto:
Se o item for PA, use código PA. Se for ME, use código ME.
EAN e Descrição:
Um EAN de cada tipo por linha. Não repita o mesmo EAN se o modelo for idêntico.
Quantidade:
Faça a somatória total de todos os equipamentos de cada EAN.
Valor Unitário:
Use o valor de uma NF do EAN (priorize os dados da 1ª NF encontrada).
Solicitação/CRG:
Preencha o campo de "Solicitação de reparo" com o número do CRG.
Notas de Aquisição:
Liste todos os números das NFs de aquisição correspondentes aos itens.
