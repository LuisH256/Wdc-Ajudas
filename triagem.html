Instruções para Triagem de Equipamentos

Siga este fluxo para realizar a triagem dos equipamentos inseridos na planilha "RELAÇÃO DE PRODUTOS WDC NETWORK".

1. Consulta no Protheus (Módulo 04)

Utilize os seriais fornecidos pelo cliente para localizar as informações de cadastro:

  • Módulo Componentes: Pesquise o serial para encontrar o Código SN e o PA.
    ➔ Clique aqui para ver o passo a passo de pesquisa de Serial

  • Módulo Produto Acabado: Use o PA coletado para verificar o EAN e Descrição. Verifique no histórico a movimentação de romaneio para identificar a Data de Garantia e NF.
    ➔ Clique aqui para ver o passo a passo de pesquisa de PA (LVP)
Regra de Cálculo Exclusiva para Protheus:
A adição de tempo abaixo aplica-se apenas para datas coletadas diretamente no Protheus:
• Garantia Padrão: Adicionar 1 ano à data encontrada.
• Se for Revendedor: Adicionar 1 ano e 3 meses.

2. Utilizando Automações (Microsoft Teams)

Visão (Unitário):

  • No chat do Visão, digite "Oi", selecione "Pesquisar Número Serial" e envie o SN.

Serial VL (Em Escala):

  • Clique com o botão direito em uma conversa > Serial VL.
  • Clique em Abrir Planilha e cole os seriais usando CTRL + SHIFT + V.
  • Após fechar e confirmar, você receberá um link com o resultado completo.
Nota sobre Automações: Para datas coletadas via automação, siga a data informada normalmente (sem somar 1 ano), com exceção apenas para produtos Fujikura.

EQUIPAMENTO FORA DE GARANTIA:
Caso esteja fora de garantia e o cliente seja REVENDEDOR:
• Adicione +3 meses extras somente se a operação for VENDA.
• Se for operação de LOCAÇÃO, não adicionar os 3 meses.

4. Organização da Planilha e NF

Padronização do Arquivo NF:

  • Realize o download da NF: ➔ Como baixar NF
  • Renomeie o arquivo adicionando os zeros à esquerda conforme a filial:
    • Filial Padrão: Adicionar 000 (Ex: 000921421).
    • Filial 08: Adicionar 0000 (Ex: 000021521).

Salvamento no SharePoint:

  • Acesse o grupo Suporte & Garantia.
  • Caminho: Documentos ➔ Notas ➔ NFs Triagem.
  • Faça o upload do arquivo padronizado nesta pasta.

Compartilhamento e Link:

  1. Clique em Compartilhar no arquivo carregado.
  2. Clique no ícone da Engrenagem ⚙️ (Configurações).
  3. Selecione a opção: "Somente pessoas com acesso existente".
  4. Clique em Aplicar e depois em Copiar link.

Vinculando na Planilha:

  • Volte para a aba TRIAGEM da sua planilha.
  • Selecione a célula da NF, clique com o botão direito e vá em "Inserir Link" ou use o atalho CTRL + K.
  • Cole o link do SharePoint e confirme.
Dica de Validação CFOP:
• 5101 / 6101: Venda
• 5949 / 6949: Locação
