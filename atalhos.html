Atalhos

Atalhos do Computador

  • Ctrl + C: Copiar
  • Ctrl + V: Colar
  • Alt + Tab: Alternar entre janelas
  • Ctrl + Shift + Esc: Abrir o Gerenciador de Tarefas
  • Windows + D: Mostrar área de trabalho

Atalhos do Excel

  • Ctrl + S: Salvar
  • Ctrl + Z: Desfazer
  • F4: Repetir última ação
  • Ctrl + Shift + L: Aplicar/Remover filtros
  • Alt + =: AutoSoma
