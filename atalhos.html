Atalhos

Atalhos do Windows (Gerais)

  • Ctrl + C Copiar
  • Ctrl + V Colar
  • Ctrl + X Cortar (Remove o item de um local e cola em outro)
  • Ctrl + Z Desfazer a última ação
  • Ctrl + Y Refazer (Volta itens removidos/desfeitos pelo Ctrl + Z)
  • Ctrl + A Selecionar todos os itens da janela ou do documento
  • Alt + Tab Alternar entre janelas abertas
  • Alt + F4 Fechar a janela ou programa atual; se estiver na área de trabalho, abre a caixa de diálogo Desligar
  • Ctrl + Shift + Esc Abrir o Gerenciador de Tarefas
  • Windows + D Mostrar/Ocultar a Área de Trabalho
  • Windows + L Bloquear o computador
  • Windows + E Abrir o Explorador de Arquivos
  • Windows + Ponto (.) ou Vírgula (,) Abrir o painel de Emojis
  • F2 Renomear o arquivo selecionado

Atalhos do Microsoft Excel

  • Ctrl + S Salvar
  • F4 Repetir última ação ou tornar referências de célula absolutas/relativas ($A$1 para A1 e vice-versa)
  • Shift + Scroll (Bolinha do mouse) Move a planilha para esquerda ou direita (Rolagem Horizontal)
  • Ctrl + Shift + L Aplicar/Remover Filtros em um intervalo
  • Alt + = AutoSoma (Insere a função `SOMA()`)
  • Ctrl + Ponto e Vírgula (;) Inserir a data atual
  • Ctrl + Shift + Ponto e Vírgula (;) Inserir a hora atual
  • Ctrl + 1 Abrir a caixa de diálogo Formatar Células
  • Ctrl + Seta Mover para a borda da região de dados atual (útil para navegar rapidamente)
  • Ctrl + Shift + Seta Selecionar dados até a borda da região atual
  • Ctrl + T Aplicar Formato de Tabela ao intervalo de dados selecionado
  • Ctrl + End Ir para a última célula usada na planilha
